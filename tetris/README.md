#Tetris 

Project Overview
This folder contains a fully functional Tetris game for the Linux terminal. The game is part of the game-room repository, aiming to provide fun and educational terminal-based games. Written in C, this version of Tetris replicates the classic gameplay in a lightweight, terminal environment.

Game Description
Tetris is a famous puzzle game where differently shaped pieces (Tetrominoes) fall from the top of the screen. The goal is to move and rotate the pieces to create complete horizontal lines, which then disappear to earn points. If the stack of pieces reaches the top, the game ends.
